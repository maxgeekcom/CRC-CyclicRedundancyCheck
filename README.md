# (CRC) - CyclicRedundancyCheck

## Description
This is the app, where you can try CRC. You can calculate or verify checksum of file or some text.

The following types of CRCs are used in app:
- CRC-4;
- CRC-8;
- CRC-16;
- CRC-32.

## Making .exe
You can also make “.exe” from source files. You have to change directory to project directory and enter the command *python setup.py build* in terminal.
